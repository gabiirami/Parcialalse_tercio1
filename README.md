Este proyecto consiste en un programa en C++ que permite contar cuántos puntos del plano cartesiano pertenecen a un cuadrante específico. El programa define una estructura Point con coordenadas x y y, implementa la función contarPuntosEnCuadrante(Point puntos[], int n, int cuadrante) para recorrer un arreglo de puntos, verificar en qué cuadrante se encuentra cada uno e ignorar aquellos ubicados sobre los ejes (x = 0 o y = 0), y retorna la cantidad correspondiente. En el main() se solicitan al usuario la cantidad de puntos, sus coordenadas y el cuadrante a evaluar, luego se llama a la función y se muestra el resultado en pantalla. El desarrollo se gestionó utilizando Git para el control de versiones y se alojó en un repositorio público en GitHub para su entrega.
